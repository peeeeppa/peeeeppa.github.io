Markdown ?

Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia、简书等。

千万不要被「标记」、「语言」吓到，Markdown的语法十分简单，常用的标记符号不超过十个，用于日常写作记录绰绰有余，不到半小时就能完全掌握。

就是这十个不到的标记符号，却能让人优雅地沉浸式记录，专注内容而不是纠结排版，达到「心中无尘，码字入神」的境界。(出自Markdown官网)

markdown 语法

1. 标题

用 # 后加一个空格来标记 能表示1-6级标题 级数越大 字号越小

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题
###### 六级标题

2. 字体

用 _ (下划线)和 *(星号)前后夹起来皆可 一个是斜体两个是粗体 三个是粗斜体

*这是斜体*

**这是粗体**

***这是粗斜体***

  • 两句话之间加一个空行
  • <br/>

4. 引用

用 > 来实现 可以嵌套

>这是引用

>>这是引用的引用

>>>这是引用的引用的引用

实际上每多一个引用只用敲多敲一个>就行了(应该

5. 链接

有两种方法:

+ \[链接名字\]\(链接地址\)
+ \<链接地址\>

[blog_home](https://peeeeppa.github.io\)

<https://peeeeppa.github.io\>

是这样的

blog_home

https://peeeeppa.github.io

6.图片

![Alternative text](link “optional title”)

​ Alternative text: 可以不写 但是必须要有中括号

​ link: 本地链接 或者 URL地址

​ “optional title”: 鼠标悬置于图片上会出现的标题文字 可以不写

![Markdown](https://img2.imgtp.com/2024/05/10/bmIfTltO.jpg “MARKEDPWN”)

7.列表

有 有序列表 和 无序列表 两种

  • 无序列表: 用 * + - 中的一种 加一个空格作为标记
    • 有序列表: 用 数字 加 . 再加一个空格做为标记

可以用 Tab 控制列表的层级

* 无序列表 1 (回车后按一下Tab)

​ 无序列表 1.1

+ 无序列表 2

- 无序列表 3

1.有序列表 1 (回车后按一下Tab)

​ 1. 有序列表 1.1

2. 有序列表 2

  • 无序列表 1
    • 无序列表 1.1
  • 无序列表 2
  • 无序列表 3
  1. 有序列表 1
    1. 有序列表 1.1 (????我明明是二级的哇?????????)
  2. 有序列表 2

8.分割线

用三个或多个 星号* 或者 减号- 官方说的是破折号 但我是减号 或者 下划线_

前后一定要记得加空行(兼容性问题)

*** --- ___

9. 删除线

分别用两个~包起来

~~我喜欢你~~

我喜欢你

10. 下划线

和HTML中相似 用<u> </u>包起来

<u>这是一个下划线示例</u>

这是一个下划线示例

10. 代码/代码块

用 反引号` 包起来 如果是代码块 就分别用三个 反引号`包起来(按Tab或者四个空格都行) 可以在第一个三反引号后面不换行写出对应语言获得高亮

` hello world `

```cpp

#include<stdio.h>

int main(){

(这里是一个Tab)printf(“hello world\n”);

}

```

hello world

11. 表格

用 | 分割单元格 用三个或多个 - 分割表头和其他行

用 :- 左对齐本列 用 -: 右对齐本列 用 :-: 居中本列

| 标题1 | 标题2 | 标题3 | 标题4 |

| ----- | :------ | :---: | ---: |

| 123 | dhakdhak | aksdg | 23167|

标题1 标题2 标题3 标题4
123 dhakdhak aksdg 23167

如果你觉得麻烦 可以使用第三方工具 Markdown Tables Generator
